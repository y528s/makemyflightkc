# Make My Flight KC

Kansas City's flight departure calculator for MCI.

Know exactly when to leave for the airport.

Free. No ads. Built by a KC local.

## What it does

Calculates exactly when you need to leave your KC neighborhood to make your MCI flight — accounting for drive time, parking, security lane, bags, party size, and KC-specific conditions including Chiefs game days.
